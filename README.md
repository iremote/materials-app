# materials-app

Demo app for visualizing materials data.

## Development

Refer to [package.json](./package.json) for project requirements (node, yarn, etc).

```bash
yarn install
yarn start
```

Launches the app in development mode at [http://localhost:3000](http://localhost:3000).
